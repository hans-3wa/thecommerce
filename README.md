# Utiliser l'application

## Docker

### Start
```bash
docker-compose up
```
### Backend
**run** to port 5300

**view**: http://localhost:5300

*this application is made with **node** && **express***

### Client
**run** to port 3000

**view**: http://localhost:3000

*this application is made with **ReactJS***

### Database
**run** to port 28017

**stringConnection**: mongodb://localhost:28017/

*Database MongoDB*

### Stop
```bash
docker-compose stop
```
